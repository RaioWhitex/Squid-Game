# 🔶 Geometric Shapes Card UI

Uma interface minimalista desenvolvida com foco em simplicidade, alinhamento visual e harmonia estética, utilizando formas geométricas básicas para demonstrar conceitos fundamentais de layout e estilização em HTML e CSS.

---

## 📸 Preview do Projeto

Interface contendo:

- Cartão centralizado na tela
- Forma circular
- Forma triangular
- Forma quadrada
- Fundo em tons quentes
- Efeito de sombra suave
- Layout limpo e organizado

---

## ✨ Tecnologias Utilizadas

- **HTML5**
- **CSS3**

---

## 🎨 Características do Design

- Interface minimalista
- Paleta monocromática em tons dourados
- Formas geométricas com contorno destacado
- Cartão com bordas arredondadas
- Sombra para efeito de profundidade
- Espaçamento equilibrado entre elementos
- Estrutura simples e elegante
- Layout responsivo

---

## 🚀 Objetivo do Projeto

Praticar conceitos essenciais de desenvolvimento front-end através da criação de uma interface visual simples, explorando alinhamento de elementos, centralização com CSS, uso de Flexbox e aplicação de estilos modernos.

---

## 📚 Funcionalidades

- Exibição de formas geométricas básicas
- Alinhamento horizontal dos elementos
- Centralização completa na tela
- Aplicação de sombras e bordas arredondadas
- Estrutura adaptável para diferentes resoluções
- Design limpo para estudos de UI
